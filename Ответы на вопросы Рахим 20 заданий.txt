1. Как правильно настроить форму для отправки данных в обработчик? Какие атрибуты прописать, какие значения?
    У формы обязательно должны быть заданы атрибуты:
    - method это метод передачи данных
    - action адрес и наименование обработчика формы например /ajax/handler.php
    У полей формы также задаем обязательно атрибут name
    У кнопки button атрибут type - submit что будет означать отправку данных формы.

2. Что такое обработчик? Какую задачу он решает? А зачем выделять его в отдельный файл? В чем польза такого подхода?
    - php скрипт в котором поступившие данные из глобального массива смотря каким методом были переданы обрабатываются и записываются в базу, отправляются на почту и т.д 
    - выделяется в отдельный файл для разделения кода и удобства дальнейшей работы

3. Какие методы бывают отправки данных через форму? В чем их разница? В каком случае использовать один, а когда другой?
    - post и get
    - отличия этих методов в способе отправки данных на сервер get параметры публичны и видны в адресной строке, а post неотображаются в адресной строке и большой объём
    - метод GET, когда нудно просто запросить данные с сервера, метод POST, когда нужно отправить данные на сервер для обработки или изменения

4. Опишите процесс записи строки в базу данных: как подключиться, подготовить запрос и выполнить
    - Для подключения к базе данных используем PDO. Объявляем переменные хост, базу данных, пользователя и пароль
        $host = 'localhost';
        $dbname = 'cg_db';
        $dbuser = 'root';
        $dbpassword = '';
    - выполняем подключение к базе создавая объект PDO ($pdo = new PDO("mysql:host=$host;dbname=$dbname", $dbuser, $dbpassword);)
    - формируем запрос ($query = 'INSERT INTO photos (name) VALUES (?)';)
    - запрос выполняем ($statement = $pdo->prepare($query);)
    - заменяем заглушки в запросе реальным значением ($statement->execute([$name]);)

5. Какой глобальный массив используется при создании флеш-сообщений?
    - Используется - $_SESSION

6. Опишите пошагово как создать и вывести флеш-сообщение
    - запускаем сессию вызывая функцию session_start();
    - создаем и заполняем элемент массива например $_SESSION["message"] = 'Неверный логин или пароль';
    - вывод echo $_SESSION["message"];
    - удаление элемента сессии unset($_SESSION['message']);

8. Какие функции используются для генерации пароля и его проверку?
    - генерация пароля password_hash($pass, PASSWORD_DEFAULT); параметры это сам пароль и константа метод шифрования.
    - password_verify() принимает 2 параметра (первый введенный пароль и второй из базы зашифрованный) и сравнивает их.

16. Какой глобальный массив хранит все файлы передаваемые через форму?

17. Опишите структуру глобального массива хранящего файлы: какие имеет ключи и значения
    - первый ключ это будет название поля типа file через которое выбирал картинки пользователь
    - далее все ключи это будут массивы если мы говорим о множественной загрузке файлов
    - name наименования загружаемых файлов
    - type тип картинки
    - tmp_name временное имя файла на серввере
    - error если во время загрузки файла произошкла какая либо ошибка
    - size размер файла в байтах

25. Объясните человеку в чем разница между $_SESSION и $_COOKIE: «Не пойму в чем разница между сессией и куки? Вроде бы одно и то же…пока полностью не осознал.»
    - глобальный массив $_COOKIE хранит данные в хранилище вашего браузера заданное программистом время, и устаревают удаляется автоматически например по истечению 30 дней. Например на сайте интернет магазина могут в $_COOKIE хранить вашу локацию город который вы выбрали.
    А сессия хранит данные только в рамках текущего соединения с сайтом. Если вы вкладку закрыли с сайтом и откроете его в новой вкладке то $_SESSION будет обновлен.
